# Quick Start Guide

## 5-Minute Setup

### 1. Create Your First Project (30 seconds)
1. Click **"Create Project"** in the left sidebar
2. Enter a name like "Work Tasks" or "Personal"
3. Click **"Create Project"**
4. **Notice the URL changes** to include your project ID (e.g., `/?project=abc-123`)
5. **Bookmark this URL** to quickly access this project later!

### 2. Add Some Tasks (1 minute)
1. Click **"New Task"** in the top right
2. Enter a task description (e.g., "Review quarterly report")
3. Optionally add:
   - Priority (High/Medium/Low)
   - Due date
   - Tags
4. Click **"Create Task"**
5. Repeat for 3-5 tasks

### 3. Try a Time Management System (2 minutes)

#### Option A: Try DIT (Simplest)
1. Find the **"Time Management System"** dropdown in the left panel
2. Select **"Do It Tomorrow (DIT)"**
3. Click **"Switch System"** in the confirmation dialog
4. Your tasks now appear in "Tomorrow"
5. **Drag a task** from Tomorrow to Today, or click the **‚Üê arrow** button
6. Focus on completing today's tasks!

**Pro tip:** You can also drag tasks to "Unscheduled" to remove them from the schedule.

#### Option B: Try AF4 (Most Intuitive)
1. Select **"Autofocus 4 (AF4)"** from the dropdown
2. Click **"Mark"** on 3-5 tasks you feel ready to work on
3. Marked tasks appear at the top
4. Work through them from top to bottom
5. Click **"Unmark"** if you change your mind

#### Option C: Try FVP (Most Systematic)
1. Select **"Final Version Perfected (FVP)"** from the dropdown
2. Click **"Start Selection"**
3. For each task, answer: "Would I do this before X?"
   - Click **"Yes, do this before X"** to prioritize it
   - Click **"Skip"** to pass
4. Click **"End Selection"** when done
5. Work through dotted tasks from top to bottom

### 4. Explore Features (1 minute)
- Click on a task to see details in the right panel
- Try the search bar to find tasks
- Use filters to show only high-priority or incomplete tasks
- Click the sun/moon icon to change themes
- **Try different view modes**: Click List/Board/Calendar buttons to switch views
  - **Board View**: Drag tasks between columns (requires columns to be set up)
  - **Calendar View**: See tasks by due date on a calendar

## Common Workflows

### Sharing Projects
**Want to share a project with someone?**
1. Open the project you want to share
2. Copy the URL from your browser's address bar (e.g., `https://your-app.com/?project=abc-123`)
3. Share the URL via email, chat, or any other method
4. The recipient can open the URL to view the same project (if they have access to your deployed app)

**Note:** Since data is stored locally, sharing a URL only works if:
- You're both using the same deployed instance, OR
- You export your data and they import it on their device

### Daily Review (DIT)
**Morning:**
1. Open the app
2. Review "Today" tasks (yesterday's incomplete tasks rolled over)
3. **Drag 2-3 tasks** from "Tomorrow" to "Today" if you have capacity (or use arrow buttons)
4. Start working from the top

**Evening:**
1. Check off completed tasks
2. Review "Tomorrow" list
3. Add any new tasks (they go to "Tomorrow" automatically)

**Pro tip:** When sections have more than 10 tasks, they become scrollable for easier navigation.

### Focus Session (AF4)
1. Review your unmarked tasks
2. Mark 5-7 tasks you feel ready to tackle
3. Set a timer for 25 minutes (Pomodoro)
4. Work through marked tasks in order
5. Take a break, then repeat

### Weekly Planning (FVP)
1. Add all tasks for the week
2. Click "Start Selection"
3. Go through the comparison process (5-10 minutes)
4. Work through dotted tasks throughout the week
5. Reset and re-prioritize mid-week if needed

## Need Help?

### Common Questions

**Q: Where is my data stored?**
A: Locally in your browser's localStorage. No data leaves your device.

**Q: Can I use this on multiple devices?**
A: Yes! Export your data on one device, then import it on another. Each device stores data locally.

**Q: Can I bookmark specific projects?**
A: Yes! Each project has its own URL (e.g., `/?project=abc-123`). Bookmark the URL to jump directly to that project.

**Q: Do browser back/forward buttons work?**
A: Yes! You can use browser navigation to move between projects and the project list.

**Q: What happens if I clear my browser data?**
A: Your tasks will be lost. Export regularly to backup.

**Q: Can I switch between time management systems?**
A: Yes! Your tasks remain unchanged, only the organization changes.

**Q: How do I backup my data?**
A: Click "Data" ‚Üí "Export Data" in the header. Save the JSON file.
